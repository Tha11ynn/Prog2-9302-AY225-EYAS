Programming 2 Machine Problem Set

Java and JavaScript CSV Dataset Processing

Students will create programs that process the provided CSV dataset in this repository. Each student must complete three (3) assigned machine problems. All programs must be implemented in both Java and JavaScript.

General Instructions

  1. Pull the repository from Git.
  2. Locate the CSV dataset included in the repository.
  3. Your program must FIRST ask the user to input the dataset file path before performing operations.
  4. Each machine problem must be implemented in:
    • Java
    • JavaScript
  5. Programs must correctly read and parse CSV files.
  6. Students must provide clear code comments explaining:
    • Variables
    • Functions
    • Processing logic
    • Dataset handling
  7. Code comments are required to make evaluation easier.
  8. Output must be properly formatted.
  9. Each submission must include:
    • Source Code
    • Screenshot of Output
    • Short explanation of program logic (3–5 sentences)

Java Program Flow

  1. Start Program
  2. Prompt user for CSV dataset file path
  3. Read file using BufferedReader and FileReader
  4. Store records using ArrayList or Objects
  5. Process dataset depending on machine problem
  6. Display formatted results
  7. Handle errors
  8. End Program

JavaScript Program Flow

  1. Start Program
  2. Prompt user for dataset path
  3. Read dataset using Node.js File System module
  4. Parse CSV into arrays or objects
  5. Process dataset according to machine problem
  6. Display formatted output
  7. Handle errors
  8. End Program

Machine Problems

MP01 – Load dataset and display total records.
MP02 – Display first 10 rows of dataset.
MP03 – Search for a keyword.
MP04 – Count valid dataset rows.
MP05 – Extract values from a column.
MP06 – Display unique values.
MP07 – Sort records alphabetically.
MP08 – Filter records by keyword.
MP09 – Display dataset statistics.
MP10 – Detect duplicate records.
MP11 – Column frequency counter.
MP12 – Display dataset as formatted table.
MP13 – Identify rows with missing data.
MP14 – Count keyword occurrences.
MP15 – Export first 50 rows to CSV.
MP16 – Random dataset sampler.
MP17 – Find longest text entry.
MP18 – Remove rows with empty fields.
MP19 – Generate dataset summary report.
MP20 – Convert CSV dataset to JSON.
